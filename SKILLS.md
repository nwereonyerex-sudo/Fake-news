# SKILLS.md

## Essential Core Machine Learning & Data Skills

Before structuring an AI agent's operational file, you need the underlying engineering competencies to train or prompt the underlying models.

### Advanced Text Preprocessing
Deep familiarity with tokenization, stopword removal, lemmatization, and handling linguistic feature engineering (e.g., emotional bias, sensational headlines).

### Word Embeddings & Vectorization
Skills in implementing TF-IDF, Bag-of-Words (BoW), GloVe, and pre-trained Transformer embeddings like RoBERTa or BERT to convert text strings into numerical tensors.

### Hybrid Deep Learning Architectures
The ability to build recurrent frameworks like LSTM-GRU or hybrid LSTM-CNN models to extract temporal structures and subtle contextual patterns from text.

### Graph Neural Networks (GNNs)
Skills in social media network analysis to track propagation patterns and identify structural biases or bot networks spreading misinformation.
